Class: dXModel

dXModel

new dXModel()

This basic model in the dexter framework is used to connect the model to the pipe event emitter network for further communication between distant views, models and collections. It provides the necessary (dis)connect methods and initializes the pipe connection on dXModel#initialize.

dXPipe

To communicate between views, distant collections and models, dexter uses an event emitter as a 'Pipe Network'. It can be (dis)connected with dXView#dXDisconnect and dXView#dXConnect.
