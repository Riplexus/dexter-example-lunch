Class: dXItem

dXItem

new dXItem()

dXItem is the representation of a collection entry in the dom. This and every extended view can be referenced to any collections view attribute and will be created if items are added to the collection.

dXEnter()

The template has to be available at the dXItem#el call to support the data binding through epoxy. Thus we have to override dXView#dXEnter to prevent the html to be replaced. We still need to set the $el id, load any subviews with dXView#dXGetSubViews and call the enter functions via dXView#dXCallEnter.
